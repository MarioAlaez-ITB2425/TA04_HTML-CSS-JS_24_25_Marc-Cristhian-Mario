Lenguaje de Marcas

En este proyecto, creé un formulario para recolectar datos, los cuales se almacenan inicialmente en formato CSV. Este formato es simple y permite guardar la información de manera tabular. Luego, convertí esos datos a XML, un formato estructurado que organiza la información de manera jerárquica.

Posteriormente, utilicé Python para procesar el archivo XML, extrayendo y mostrando los datos de manera dinámica. Finalmente, los datos procesados se almacenaron en formato JSON, que es ligero y fácil de usar para futuras consultas o integraciones con otros sistemas. Este flujo de trabajo optimiza la recolección, el procesamiento y el almacenamiento de información de forma eficiente.
